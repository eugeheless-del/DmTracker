# 📢 Telegram Broadcast Feature

Система массовой рассылки уведомлений через Telegram для DM Tracker.

## 🚀 Быстрый старт

### 1. Получение Telegram Bot Token

1. Откройте Telegram и найдите бота [@BotFather](https://t.me/botfather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям BotFather для создания бота
4. Скопируйте полученный token (выглядит как: `123456789:ABCdEfGHIJKlmnOPqrSTuvWxYZ`)

### 2. Добавление Chat ID для персонажей

Каждому персонажу необходимо присвоить Telegram Chat ID (ID чата, куда будут отправляться сообщения):

1. Откройте Dashboard → 🧙 Персонажи
2. Отредактируйте персонажа
3. В поле **Telegram Chat ID** введите ID

**Как узнать свой Chat ID:**
- Отправьте боту сообщение `/start`
- Используйте инструмент вроде https://t.me/userinfobot
- Chat ID будет выглядеть как число, например: `123456789`

### 3. Настройка Bot Token в приложении

Token можно установить одним из способов:

**Способ 1: Через браузер (localStorage)**
```javascript
// Откройте консоль браузера (F12) и выполните:
localStorage.setItem('telegram_bot_token', 'ВАШ_TOKEN_ЗДЕСЬ');
```

**Способ 2: Через код приложения**
- Создайте файл `.env.local` в корне проекта:
```
VITE_TELEGRAM_BOT_TOKEN=ВАШ_TOKEN_ЗДЕСЬ
```

## 📋 Использование

### Отправка сообщений

1. Нажмите кнопку **📢 Telegram** на Dashboard
2. Откроется модальное окно с опциями:

### Функции модального окна

#### 🔍 Поиск игроков
- Введите имя персонажа или игрока для фильтрации
- Ищет по обоим полям в реальном времени

#### 👥 Выбор персонажей
- **Чекбоксы** — выберите конкретных игроков
- **Выбрать все / Снять все** — быстрый выбор
- **❌ Нет ID** — персонажи без Telegram Chat ID не отмечены

#### 💬 Сообщение
- Поддерживает **Markdown** формат:
  - `*жирный текст*` → **жирный текст**
  - `_курсив_` → _курсив_
  - `` `код` `` → `код`
- **Максимум 4096 символов** (ограничение Telegram API)
- Счётчик символов показывает текущий размер

#### 📤 Отправка

Три варианта отправки:

1. **✉️ Отправить выбранным** — только отмеченные персонажи
2. **📢 Отправить ВСЕМ** — игнорирует выделение, отправляет всем
3. **Отмена** — закрыть окно без отправки

### 📊 Процесс отправки

Во время отправки:
- Показывается **индикатор прогресса** ("Отправлено 3 из 10...")
- Между сообщениями добавляется **задержка 150ms** для соблюдения API лимитов
- Если бот заблокирован, ошибка не прерывает рассылку

### 📈 Отчёт о результатах

После завершения отправки показывается:
- ✅ **Успешно** — количество доставленных сообщений
- ❌ **Ошибок** — количество неудачных отправок
- **Список ошибок** — перечень проблем с указанием причины:
  - "Bot blocked by user" — бот заблокирован
  - "Invalid chat ID" — неверный Chat ID
  - "Failed to fetch" — проблема сети

### Функции в `telegram.ts`

```typescript
// Отправка одного сообщения
sendTelegramMessage(chatId: string, text: string): Promise<Result>

// Массовая рассылка с прогрессом
sendTelegramBroadcast(recipients: Array<{...}>, text: string, onProgress): Promise<Result[]>

// Генерация отчёта
generateBroadcastReport(results): {successful, failed, failedList}
```

### Форматирование сообщений

Используется формат **Markdown** (указано в API вызове):
```typescript
parse_mode: 'Markdown'
```

Поддерживаемые теги:
- `*текст*` — жирный
- `_текст_` — курсив
- `` `текст` `` — моноширинный шрифт
- `[текст](URL)` — ссылки

## ⚠️ Важные замечания

### Безопасность
- **Не делитесь Bot Token** — он может быть использован для управления вашим ботом
- **Не коммитьте token** в Git — используйте `.env.local`
- Chat ID — это публичная информация, безопасна для хранения

### Ограничения Telegram API
- **4096 символов** максимум на сообщение
- **~30 сообщений в секунду** на бота (соблюдается через задержку)
- **24 часа** на восстановление после блокировки бота

### Технические требования
- PC должен иметь поле `telegram_chat_id` заполненным
- Bot Token должен быть валидным и активным
- Интернет соединение требуется

## 🐛 Решение проблем

### "Персонажей не найдено"
- Убедитесь, что добавили персонажей в разделе 🧙 Персонажи
- Проверьте, что у персонажей заполнено поле Telegram Chat ID

### "Отправить ВСЕМ" кнопка неактивна
- Убедитесь, что есть персонажи с заполненным Telegram Chat ID
- Проверьте, что введено сообщение

### Сообщения не отправляются
1. Проверьте Bot Token в консоли браузера: `localStorage.getItem('telegram_bot_token')`
2. Убедитесь, что бот не был заблокирован
3. Проверьте Chat ID на корректность (должны быть только цифры)
4. Откройте консоль браузера (F12) и проверьте ошибки

### Бот не отвечает на сообщения
- Убедитесь, что бот запущен на [@BotFather](https://t.me/botfather)
- Проверьте, что добавили бота в чат
