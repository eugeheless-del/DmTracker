### 1. ✅ Telegram Utility функции
**Файл:** [src/utils/telegram.ts](src/utils/telegram.ts)

- `sendTelegramMessage()` — отправка одного сообщения
  - Обработка ошибок (заблокирован бот, невалидный ID)
  - Парсинг Markdown формата
  - Правильное отображение ошибок

- `sendTelegramBroadcast()` — массовая рассылка
  - Задержка 150ms между сообщениями (API лимиты)
  - Callback прогресса в реальном времени
  - Раздельная обработка каждого получателя

- `generateBroadcastReport()` — генерация отчёта
  - Подсчёт успешных/неудачных отправок
  - Список ошибок с указанием причин

## 🚀 Как использовать

### Первоначальная настройка

1. **Получить Bot Token:**
   ```
   → Telegram: @BotFather
   → /newbot
   → Скопировать token
   ```

2. **Установить token в браузер:**
   ```javascript
   // Консоль браузера (F12):
   localStorage.setItem('telegram_bot_token', 'ВАШ_TOKEN');
   ```

3. **Добавить персонажей:**
   - 🧙 Персонажи → Игрок
   - Заполнить Telegram Chat ID
   - Сохранить

### Использование

1. На Dashboard → нажать **📢 Telegram**
2. Выбрать персонажей или поискать
3. Написать сообщение (Markdown поддерживается)
4. Нажать "Отправить выбранным" или "Отправить ВСЕМ"
5. Дождаться завершения и увидеть отчёт

## 🔧 Технические детали

### Стек
- React 18+ (функциональные компоненты, хуки)
- TypeScript (строгая типизация)
- Tailwind CSS (стили, адаптивность)
- Zustand (управление состоянием)
- Fetch API (HTTP запросы)

### Ключевые особенности
- **Безопасность:** Bot Token хранится в localStorage
- **Производительность:** Задержка между запросами для API лимитов
- **UX:** Реальный прогресс, подробные ошибки
- **Надёжность:** Обработка сетевых ошибок, retry-логика

### Форматирование Telegram сообщений
```
*жирный текст*       → **жирный текст**
_курсив_             → _курсив_
`код`                → `код`
[ссылка](https://...) → ссылка
```

## 📊 Поддерживаемые сценарии

✅ **Успешная отправка** — зелёный галочка в отчёте
❌ **Бот заблокирован** — ошибка обработана, рассылка продолжается
❌ **Невалидный Chat ID** — персонаж пропущен
❌ **Проблемы сети** — показывается сообщение об ошибке
