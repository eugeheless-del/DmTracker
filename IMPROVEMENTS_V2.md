# DM Tracker v2 - Рекомендации по улучшениям

## Текущее состояние
- ✓ Глобальный поиск реализован (имя, роль, тип, описание)
- ✓ SearchBar в шапке с выпадающими результатами
- ✓ localStorage корректно сохраняет/восстанавливает данные
- ✓ React-ворнинги очищены (type guard в фильтрах)

---

## v2 Приоритеты

### 1. **Фильтрация и сортировка** [High]
- [ ] Фильтр по статусу твиста на странице Твистов (уже есть + улучшить)
- [ ] Сортировка персонажей (по уровню, классу)
- [ ] Сортировка сессий (по дате, последние первыми по умолчанию)
- [ ] Мультиселект для фильтрации (выбор нескольких статусов одновременно)

### 2. **Связи между сущностями** [High]
- [ ] Страница связей (граф или таблица): какие твисты связаны с персонажами
- [ ] Подсчет: сколько твистов/сессий на персонажа
- [ ] Быстрое удаление персонажа (удалить и его связи)
- [ ] Визуальный индикатор: персонаж используется в N сессиях/твистах

### 3. **UI/UX улучшения** [Medium]
- [ ] Темная тема: добавить переключатель светлой/темной темы
- [ ] Модальные окна: добавить анимацию (fade-in)
- [ ] Карточки: выбрана, то подсветить (для мобильного редактирования)
- [ ] Breadcrumbs навигация (где пользователь находится)
- [ ] Подтверждение удаления: модаль вместо alert

### 4. **Поиск v2** [Medium]
- [ ] Поиск с фильтром по типам (радио "Все / ПЛ / НПЛ / Твисты / Сессии")
- [ ] Поиск по тегам (добавить поле tags в PC/NPC)
- [ ] Сохраненные поиски (сохранить фильтр в localStorage)
- [ ] История поиска (последние 5 запросов)

### 5. **Редактирование** [Medium]
- [ ] Inline-редактирование имени (клик на карточке = быстрое редактирование имени)
- [ ] Дублирование персонажа (скопировать все поля)
- [ ] Массовое редактирование (выбрать несколько персонажей, изменить статус)
- [ ] Отмена последнего действия (Undo)

### 6. **Импорт/Экспорт** [Medium]
- [ ] Экспорт в JSON (скачать все данные)
- [ ] Импорт из JSON (загрузить существующую кампанию)
- [ ] Экспорт в PDF (персонажи с описанием)
- [ ] Экспорт сессии как Markdown

### 7. **Интеграции** [Low]
- [ ] Синхронизация с облаком (Google Drive, Dropbox)
- [ ] Шаринг кампании (URL со встроенным кодом)
- [ ] Dark mode по умолчанию (если OS в dark mode)
- [ ] PWA (работает оффлайн)

### 8. **Данные и статистика** [Low]
- [ ] Статистика: сколько персонажей, твистов, сессий
- [ ] График активности (когда были сессии)
- [ ] Статус твистов: pie chart (скрытые/готовые/открытые/завершенные)
- [ ] Время, потраченное в игре (сумма дат сессий)

### 9. **Опции и настройки** [Low]
- [ ] Settings страница (язык, размер шрифта, тема)
- [ ] Резервную копию данных в localStorage (с таймстампом)
- [ ] Восстановление из резервной копии
- [ ] История изменений (кто и когда менял что)

### 10. **Производительность** [Low]
- [ ] Виртуализация длинных списков (если > 500 персонажей)
- [ ] Ленивая загрузка картинок (если добавим аватары)
- [ ] Кэширование поисков
- [ ] Оптимизация re-renders (React.memo для карточек)
