# UI Style Guide

## Цветовая палитра

- `--color-primary: #5680E9` — основной цвет интерфейса, используется для ключевых кнопок, заголовков и действий.
- `--color-secondary: #84CEEB` — вторичный цвет, используется для фоновых акцентов, мягких кнопок и подсветки.
- `--color-accent: #5AB9EA` — акцентный цвет для важных элементов, ссылок и подсказок.
- `--color-muted: #C1C8E4` — приглушённый цвет для фоновых блоков, меток и вторичного текста.
- `--color-purple: #8860D0` — цвет для выделения, тегов и важных статусов.
- `--color-bg-dark: #0f172a` — базовый тёмный фон страницы.
- `--color-card-bg: rgba(30, 41, 59, 0.8)` — фон карточек и панелей.

## Когда использовать каждый цвет

- `--color-primary` — главные CTA, активные элементы, заголовки и важные кнопки.
- `--color-secondary` — декоративные элементы, фоновые полосы, вторичные кнопки.
- `--color-accent` — состояния, подсказки, наведения и выделение важных деталей.
- `--color-muted` — фоны форм, подсказки, вспомогательный текст и границы.
- `--color-purple` — специальные теги, статусы, визуальные акценты с другим тоном.
- `--color-bg-dark` — общий фон приложения.
- `--color-card-bg` — панели, карточки и модальные окна.

## Сетка отступов (8px)

Мы используем базовую систему 8px: `8px`, `16px`, `24px`, `32px`, `40px`.

- `--space-xs: 0.25rem` → 4px
- `--space-sm: 0.5rem` → 8px
- `--space-md: 1rem` → 16px
- `--space-lg: 1.5rem` → 24px
- `--space-xl: 2rem` → 32px

> Используй `gap`, `padding` и `margin` кратными шагами из этой системы.

## Размеры шрифтов

- Заголовок 1: `clamp(2rem, 5vw, 3rem)`
- Заголовок 2: `clamp(1.75rem, 4vw, 2.5rem)`
- Заголовок 3: `clamp(1.4rem, 3vw, 2rem)`
- Заголовок 4: `1.25rem`
- Текст: `1rem`
- Мелкий текст: `0.85rem`

## Примеры классов

### Кнопки

- `.btn` — базовый стиль кнопки.
- `.btn--primary` — основная кнопка.
- `.btn--secondary` — вторичная кнопка.
- `.btn--accent` — акцентная кнопка.
- `.btn--danger` — кнопка удаления/отмены.
- `.btn--success` — кнопка подтверждения.

### Карточки

- `.card` — базовая карточка с фоном `--color-card-bg`, радиусом `12px` и тенью.
- `.card--bordered` — карточка с выделенной рамкой.
- `.card__title` — заголовок карточки.
- `.card__body` — текст внутри карточки.

### Формы

- `.form-control` — контейнер поля формы.
- `.input`, `.textarea`, `.select` — стили полей ввода.
- `.label` — подпись поля.
- `.input__hint` — вспомогательный текст.

## Утилиты

- `.flex`, `.inline-flex`, `.grid`, `.flex-center`, `.flex-between`, `.flex-col`, `.flex-wrap`
- `.grid-cols-2`, `.grid-cols-3`, `.grid-cols-4`
- Отступы: `.m-1`, `.mt-2`, `.p-3`, `.pl-4` и т.д.
- Цвет текста: `.text-primary`, `.text-secondary`, `.text-accent`, `.text-muted`, `.text-purple`.
- Фоны: `.bg-primary`, `.bg-secondary`, `.bg-accent`, `.bg-muted`, `.bg-card`.

## Правила

- Все новые страницы и UI-блоки создаются только в этом стиле.
- Не добавляй новые цветовые палитры вне `new-design-system.css` без согласования.
- Новые компоненты должны использовать существующие переменные и утилиты.
- Старые стили оставляем для совместимости, но новые страницы строятся на новой дизайн-системе.
