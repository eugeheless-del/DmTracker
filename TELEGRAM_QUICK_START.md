# 🚀 Telegram Broadcasting - Быстрый старт

> Реализована полная система массовой рассылки Telegram уведомлений с UI для выбора персонажей, ввода сообщений и отслеживания результатов.

## 📦 Что включено

- ✅ Кнопка на Dashboard (📢 Telegram)
- ✅ Модальное окно с выбором игроков
- ✅ Поиск и фильтрация персонажей
- ✅ Поле для ввода сообщения (Markdown поддержка)
- ✅ Две опции отправки (выбранным / всем)
- ✅ Индикатор прогресса
- ✅ Подробный отчёт результатов
- ✅ Обработка ошибок (заблокирован, невалидный ID)
- ✅ Поле Telegram Chat ID в форме персонажа

## 🔧 Первоначальная настройка (5 минут)

### 1️⃣ Создайте Telegram бота

```
1. Откройте Telegram: t.me/BotFather
2. Отправьте: /newbot
3. Следуйте инструкциям
4. Скопируйте token (123456789:ABCdEfGHIJ...)
```

### 2️⃣ Установите token в браузер

Откройте DevTools (F12) консоль и выполните:

```javascript
localStorage.setItem('telegram_bot_token', 'ВАШ_ТОКЕН_ЗДЕСЬ');
```

### 3️⃣ Получите Chat ID для игроков

Каждый игрок должен:
1. Написать боту `/start`
2. Использовать инструмент https://t.me/userinfobot (прошлёт ID)
3. Сообщить вам свой Chat ID

### 4️⃣ Добавьте Chat ID в приложение

- Откройте 🧙 Персонажи
- Отредактируйте персонажа
- Заполните поле **📱 Telegram Chat ID**
- Сохраните

## 📱 Как использовать

### На Dashboard

1. Нажмите кнопку **📢 Telegram** (верхний правый угол)
2. Откроется модальное окно

### В модальном окне

**Шаг 1: Выбрать персонажей**
- Типизируйте имя для поиска (опционально)
- Отметьте нужных игроков через ☑️
- Или нажмите "Выбрать все"

**Шаг 2: Написать сообщение**
- Введите текст в поле 💬
- Поддерживается Markdown: `*жирный*`, `_курсив_`, `` `код` ``
- Увидите предпросмотр ниже

**Шаг 3: Отправить**
- **✉️ Отправить выбранным** — только отмеченные
- **📢 Отправить ВСЕМ** — всем в базе
- **Отмена** — закрыть без отправки

**Шаг 4: Результаты**
- Увидите прогресс (Отправлено 3 из 10...)
- После завершения — отчёт:
  - ✅ Успешно: 10
  - ❌ Ошибок: 0

## ❓ FAQ

**Q: Где найти свой Chat ID?**
A: Напишите боту `/start` или используйте @userinfobot

**Q: Что делать, если сообщение не дошло?**
A: 
- Проверьте, верен ли Chat ID (только цифры)
- Убедитесь, что бот не заблокирован
- Проверьте интернет соединение
- Посмотрите отчёт об ошибке

**Q: Можно ли редактировать сообщение после отправки?**
A: Нет, но можно отправить исправленное заново

**Q: Есть ли лимит на размер сообщения?**
A: Да, максимум 4096 символов

**Q: Как часто можно отправлять сообщения?**
A: 30 сообщений в секунду на бота

## 🎯 Готовые команды для разработки

### Просмотр Bot Token
```javascript
console.log(localStorage.getItem('telegram_bot_token'));
```

### Очистка локального хранилища
```javascript
localStorage.removeItem('telegram_bot_token');
```

### Тестовая отправка через API
```javascript
// В консоли браузера
fetch('https://api.telegram.org/bot123456789:ABCdEf.../sendMessage', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    chat_id: '123456789',
    text: 'Привет!',
    parse_mode: 'Markdown'
  })
}).then(r => r.json()).then(d => console.log(d));
```

## 🐛 Решение проблем

### Кнопка не видна
- Убедитесь, что вы на странице Dashboard (📊 Общее)
- Перезагрузите страницу (Ctrl+R)

### Модальное окно не открывается
- Проверьте консоль браузера (F12 → Console)
- Убедитесь, что в localStorage сохранены персонажи

### Сообщение не отправляется
- Проверьте Bot Token: `localStorage.getItem('telegram_bot_token')`
- Убедитесь, что Chat ID содержит только цифры
- Проверьте, что бот не был заблокирован пользователем

### "Персонажей не найдено"
- Добавьте персонажей в 🧙 Персонажи
- Убедитесь, что заполнено Telegram Chat ID
